# HR-Analytics

HR Analytics project analyzing employee data to identify attrition trends ,workforce distribution   and hiring insights using MYSQL ,Advance Excel , Power BI ,and Tableau Dashboards for data-driven HR decision making.


## Project Overview

This project analyzes employee data to understand workforce trends and employee attrition. The dashboard helps HR teams monitor workforce distribution and make data-driven decisions for better employee retention and organizational planning.

## Tools Used

MySQL, Advanced Excel, and Power BI were used for data cleaning, analysis, and visualization.

## Key Insights

The analysis provides insights into employee attrition by department, gender distribution across roles, and hiring trends over time.

## Conclusion

The HR Analytics dashboard helps organizations understand employee behavior and supports HR teams in improving workforce management and retention strategies.
